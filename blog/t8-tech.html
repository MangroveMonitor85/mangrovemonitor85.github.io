Programming languages are commonly categorized as statically typed and dynamically typed. A dynamically typed language can change the variable type on the fly such as Ruby, Python and JavaScript. A statically typed language means that once a variable's type has been set, it cannot be changed to another type such as Java, C++ and Cobol. The two languages that we've been using are dynamically typed languages. Scripting languages are almost always dynamically typed languages. Functional programming concepts tend to be associated with dynamically typed languages as well.

In a statically typed language, when you create a variable in a statically typed language, you need to specify the variable type in the variable declaration.

char letter = 'a';
int wholeNumber = 10;
string sentence = "statically typed languages";
float decimalNumber = 5.5;

I have a limited understanding of Java and C++, both of which are statically typed languages. If I needed to convert one variable type to another variable type, such as int to float, I had to create two variables. One variable held the int and the other held the conversion from int to float. When dealing with legacy code in these languages, it can be a real pain trying to discover what all of the variables do, especially if they aren't named properly. In Ruby and JavaScript it is relatively easy to change the variable type, in statically typed languages it's a real pain in the ass. There is, however, a lower chance of error because there isn't any confusion in the compiler of what kind of variable holds what.

In JavaScript, the interpreter automatically sets the variable type, not when it's declared, but when it's actually initialized.

var count; // The interpreter still has not assigned a variable type
count = 0; // Now the interpreter has assigned a variable type

Ruby works exactly the same way as JavaScript in regards to variable declaration and initialization.

count; // The interpreter still has not assigned a variable type
count = 0; // Now the interpreter has assigned a variable type
