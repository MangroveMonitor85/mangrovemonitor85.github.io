Enumerable Methods

The Enumerable methods in Ruby are a set of methods that iterate through, search, sort and manipulate a collection of data. Enumerable methods can also be chained allowing several methods to work on the same object in the same line, instead of giving each method it's own line. These advantages allow code to be much more concise, elegant and readable, giving Ruby it's reputation for "glittery" code. Let's take a look below at two code snippets that accomplish the same function. On the bottom-left is a standard for loop iterating through an entire array and on the bottom-right is the enumerable method #each.

for x in 0...array.length

puts array[x]

end

array.each { |x| puts x }
