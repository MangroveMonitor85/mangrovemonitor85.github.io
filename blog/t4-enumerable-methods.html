Enumerable Methods

The Enumerable methods in Ruby are a set of methods that traverse, search, sort and manipulate a collection of objects (arrays, hashes). Enumerable methods can also be chained allowing several methods to work on the same object in the same line, instead of giving each method it's own line. These advantages allow code to be much more concise, elegant and readable, giving Ruby it's reputation for "glittery" code. Let's take a look below at two code snippets that accomplish the same function.

for x in 0...array.length

puts array[x]

end

array.each { |x| puts x }

On the top-left is a standard for loop iterating through an entire array and on the top-right is the Enumerable method #each that does the exact same. It almost seems like the for loop is obsolete because the enumerable method is much easier to read and obviously much more elegant. There are a few isolated instances where loops are used, but in Ruby these enumerable methods are vastly preferred.

The Enumerable#map method accepts a block of code and manipulates each element therein. The methods #map and #collect are exactly the same method except with a different name. The #map method returns a new array filled with whatever the block returned and #map! returns the mutated original array.

The Enumerable#group_by method essentially breaks and regroups a collection of objects into different groups based on some user-given criteria.
