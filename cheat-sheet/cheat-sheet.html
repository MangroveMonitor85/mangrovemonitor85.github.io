Ruby For Loops

Basic Ruby for loop:

for i in 0..5 // loops over values 0 through 5.
     prints "#{i} "
end
=> 0 1 2 3 4 5

JavaScript For Loops

Example of JavaScript for loop:

for (i = 0; i < cars.length; i++) {
     text += cars[i] + "";
}

Operational view of the JavaScript for loop:


for (statement 1; statement 2; statement 3) {
code block to be executed
}

Ruby Iterator Methods

Three methods that are commonly used in Ruby to iterate through a collection of objects are the #each, #collection and #map method. The #each method can be used with arrays or hashes. The #map method is, however, an enumerable method that can only be used with arrays. The #collect does the exact same thing as the #map method and they can both be used interchangeably.

my_hash = [0, 1, 2, 3, 4, 5] // our collection of objects
my_hash.map { |x| x ** x} // iterates through the array
=> [0, 1, 4, 9, 16, 25]

my_hash.collect { |x| x ** x} // does the exact same as the #map method
=> [0, 1, 4, 9, 16, 25]

my_hash.each { |x| x ** x} // It goes through the entire collection and squares all of the values
=> 0, 1, 4, 9, 16, 25

A variation on the #each theme is the #reverse_each method which iterates through the array or hash backwards.

my_hash.reverse_each { |element| x ** x } // iterates backward through the array
=> [25, 16, 9, 4, 1, 0]

The above methods are great if you want to iterate through a hash or an array on a single line, but if you need multiple lines in your block, you can use the do keyword.

my_hash = [0, 1, 2, 3, 4, 5] // our collection of objects

my_hash.each do |element|
     if element % 2 == 0
          print "#{element} "
     end
end
=> 0, 2, 4

Ruby Loops

The while, until, and for loops are loops that we have most likely used in Ruby so far in Phase 0. The while and until loops are based on some kind of condition and run as long as the condition or conditions remain true. You must be careful, however, with these two loops because it's easy to create an infinite loop with them.

The example below is the while loop which is based on fulfilling a certain condition. In the example below, the iterator, x, will continue to loop through myHash as long as its less than the value of myHash's length. Once it equals myHash's length, the condition is false and resumes the flow of control elsewhere in the program.

my_hash = [0, 1, 2, 3, 4, 5] // our collection of objects

while x < my_hash.length // Keeps looping through the hash until the condition is false
     x = 0
     print "#{my_hash[x]} "
     x += 1 // increments during every iterations
end
=> 0 1 2 3 4 5

Another loop that's commonly used in ruby is until. Until is essentially the exact same thing as the while loop except the until loop breaks once the condition is true. One must be very careful with the logic in these loops because it is very easy to make it an infinite loop. These two loops are mainly used to traverse arrays and hashes.


item = 50
until item == 0
     print "#{item} "
     item -= 10
end
=> 50 40 30 20 10

The last loop that we will cover is the for loop. This loop is usually used to iterate through integers in other programming languages and is definitely a workhorse in many C based languages such as C, C++ and Java. It's also common in JavaScript as well. When I was learning Java and C++, I can't tell you how many times I used a for loop. I used it in almost every single program I wrote in those languages. However, it doesn't seem commonly used in Ruby

my_hash = [0, 1, 2, 3, 4, 5] // our collection of objects

for x in my_hash
     print "#{x} "
end
=> 0, 1, 2, 3, 4, 5

The for loop doesn't have to be used with a collection of objects, it can be used with a range of numbers instead.

for x in 1..5
     print "#{x} "
end
=> 1, 2, 3, 4, 5

For loops can be used to count downwards (decrementing) as well, not just upwards.

for x in (50).downto(10) // The syntax is a little different when decrementing
     print "#{x} "
end
=> 50, 40, 30, 20, 10
